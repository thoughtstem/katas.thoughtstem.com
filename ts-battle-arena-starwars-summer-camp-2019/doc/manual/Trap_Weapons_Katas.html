6 Trap Weapons Katas

6.1 Spike Mine Kata (difficulty = 2)

read
Make a game that has a weapon that deploys a default spike mine.
code
#lang battlearena-starwars
 
(starwars-game
  #:weapon-list (list
   (custom-weapon
    #:name "Spike Mine"
    #:sprite (make-icon "SM")
    #:dart (spike-mine-builder))))

6.2 Spike Mine Kata (difficulty = 3)

Tip: Default #:speed is 5 and default #:range is 20.

read
Make a game that has a weapon that deploys a spike mine with a large range and speed.
code
#lang battlearena-starwars
 
(starwars-game
  #:weapon-list (list
   (custom-weapon
    #:name "Spike Mine"
    #:sprite (make-icon "SM" 'gray)
    #:dart (spike-mine-builder #:speed 10 #:range 50))))

6.3 Spike Mine Kata (difficulty = 4)

Tip: Default #:damage is 2.

read
Make a game that has a weapon that deploys a spike mine with a large range and speed, high damage, and custom sprite.
code
#lang battlearena-starwars
 
(starwars-game
  #:weapon-list (list
   (custom-weapon
    #:name "Spike Mine"
    #:sprite (make-icon "SM" 'gray 'white)
    #:dart (spike-mine-builder
     #:sprite (set-sprite-color 'red spike-mine-sprite)
     #:damage 100
     #:speed 10
     #:range 50))))

6.4 Lightsaber Droid Kata (difficulty = 2)

read
Make a game that has a weapon that spawns a droid with a lightsaber.
code
#lang battlearena-starwars
 
(starwars-game
  #:weapon-list (list
   (custom-weapon
    #:name "Lightsaber Droid"
    #:sprite (make-icon "LD")
    #:dart (lightsaber-droid))))

6.5 Lightsaber Droid Kata (difficulty = 4)

read
Make a game that has a defined weapon that spawns a droid with a customized lightsaber.
code
#lang battlearena-starwars
 
(define (ls-droid)
   (custom-weapon
    #:name "Lightsaber Droid"
    #:sprite (make-icon "LS")
    #:rarity 'epic
    #:dart (lightsaber-droid
     #:color "orange"
     #:damage 25
     #:durability 30)))
 
(starwars-game #:weapon-list (list (ls-droid)))

6.6 Lightsaber Droid Kata (difficulty = 5)

read
Make a game that has a defined weapon that spawns a droid with a fully customized lightsaber.
code
#lang battlearena-starwars
 
(define (ls-droid)
   (custom-weapon
    #:name "Lightsaber Droid"
    #:sprite (make-icon "LS")
    #:rarity 'epic
    #:dart (lightsaber-droid
     #:color "orange"
     #:damage 25
     #:speed 10
     #:durability 30
     #:range 20
     #:fire-rate 0.5)))
 
(starwars-game #:weapon-list (list (ls-droid)))
