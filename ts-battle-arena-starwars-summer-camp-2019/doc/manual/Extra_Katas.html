7.1 Planet Kata (difficulty = 1)

read
Make a game that has a random colored planet.
code
#lang battlearena-starwars
 
(starwars-game #:planet (custom-planet))

7.2 Planet Kata (difficulty = 1)

Tip: See other bg-image options in the assets library.

read
Make a game that uses a predefined image as the planet.
code
#lang battlearena-starwars
 
(starwars-game
  #:planet (custom-planet #:img LAVA-BG))

7.3 Planet Kata (difficulty = 2)

Tip: Rows and columns default to 3x3.

read
Make a game that has a fully customized planet.
code
#lang battlearena-starwars
 
(define (my-bg)
   (custom-planet
    #:img LAVA-BG
    #:rows 2
    #:columns 2))
 
(starwars-game #:planet (my-bg))

7.4 Planet Kata (difficulty = 3)

Tip: Start-tile can be between 0 (the default) and total-tiles minus 1. For example, a 2x2 grid has 4 tiles, so the last tile is tile 3.

read
Make a game that has a high definition lava background, 2 by 2 grid, and starts on tile 3.
code
#lang battlearena-starwars
 
(define (my-bg)
   (custom-planet
    #:img LAVA-BG
    #:rows 2
    #:columns 2
    #:start-tile 3
    #:hd? #t))
 
(starwars-game #:planet (my-bg))

7.5 Level Design Kata (difficulty = 1)

Tip: This will fill the world with two types of trees.

read
Make a game with a forest background filled with trees.
code
#lang battlearena-starwars
 
(starwars-game
  #:planet (custom-planet #:img FOREST-BG)
  #:enable-world-objects? #t)

7.6 Level Design Kata (difficulty = 2)

Tip: Setting hd? #t will use more detailed assets but it will take longer to load.

read
Make a game with a forest background filled with high definition trees.
code
#lang battlearena-starwars
 
(starwars-game
  #:planet (custom-planet #:img FOREST-BG)
  #:other-entities (make-world-objects
   round-tree
   pine-tree
   #:hd? #t))

7.7 Level Design Kata (difficulty = 2)

Tip: make-world-objects works best with houses or trees and must only take two types. See the Assets Library for more options.

read
Make a game with a pink background filled with random color candy-cane-trees and snow-pine-trees.
code
#lang battlearena-starwars
 
(starwars-game
  #:planet (custom-planet #:img PINK-BG)
  #:other-entities (make-world-objects
   candy-cane-tree
   snow-pine-tree
   #:hd? #t
   #:random-color? #t))

7.8 Level Design Kata (difficulty = 4)

Tip: With a 3x3 bg, the posn x value can be between 0 and 480 and the y between 0 and 360. Hue is any value between 0 and 360.

read
Make a game with 3 world objects with customized position, tile, size, and/or hue.
code
#lang battlearena-starwars
 
(starwars-game
  #:other-entities (pine-tree (posn 100 200) #:tile 0)
  (wood-house (posn 100 200) #:tile 1 #:size 0.5)
  (brick-house
   (posn 100 200)
   #:tile 2
   #:hue (random 360)))

7.9 Boost Kata (difficulty = 3)

Tip: You can also try set-damage-to and multiply-damage-by.

read
Make a game that has a powerup that temporarily boosts your damage.
code
#lang battlearena-starwars
 
(starwars-game
  #:item-list (list
   (custom-item
    #:name "Damage Boost"
    #:sprite (make-icon "DB" 'orangered)
    #:on-use (change-damage-by 1000 #:for 200))))

7.10 Boost Kata (difficulty = 3)

Tip: You can also try set-speed-to and change-speed-by. The default movement speed is 10.

read
Make a game that has a powerup that temporarily boosts your speed.
code
#lang battlearena-starwars
 
(starwars-game
  #:item-list (list
   (custom-item
    #:name "Speed Boost"
    #:sprite (make-icon "SB" 'yellow)
    #:on-use (multiply-speed-by 2 #:for 200))))

7.11 Boost Kata (difficulty = 7)

read
Make a game that has both a Damage and a Speed Boost defined powerup.
code
#lang battlearena-starwars
 
(define (damage-boost)
   (custom-item
    #:name "Damage Boost"
    #:sprite (make-icon "DB" 'orangered)
    #:on-use (change-damage-by 1000 #:for 200)
    #:rarity 'epic))
 
(define (speed-boost)
   (custom-item
    #:name "Speed Boost"
    #:sprite (make-icon "SB" 'yellow)
    #:on-use (multiply-speed-by 2 #:for 200)
    #:rarity 'uncommon))
 
(starwars-game
  #:item-list (list (damage-boost) (speed-boost)))

7.16 Lava Pit Kata (difficulty = 3)

Tip: Default #:damage is 10 and default #:size is 1.

read
Make a game that has a custom weapon that deploys a lava pit with custom damage and size.
code
#lang battlearena-starwars
 
(starwars-game
  #:weapon-list (list
   (custom-weapon
    #:name "Lava Pit"
    #:sprite (make-icon "LP" 'red)
    #:dart (lava-builder #:damage 25 #:size 2))))

7.17 Lava Pit Kata (difficulty = 5)

Tip: Default #:range is 5.

read
Make a game that has a custom weapon that deploys a lava pit with custom damage, size, sprite, and range.
code
#lang battlearena-starwars
 
(define (my-lava-pit)
   (custom-weapon
    #:name "Lava Pit"
    #:sprite (make-icon "LP" 'red 'white)
    #:dart (lava-builder
     #:damage 25
     #:size 1
     #:sprite (square 10 'solid 'black)
     #:range 10)))
 
(starwars-game
  #:weapon-list (list (my-lava-pit)))

7.19 Blaster Droid Kata (difficulty = 4)

read
Make a game that has a defined weapon that spawns a droid with a customized blaster.
code
#lang battlearena-starwars
 
(define (b-droid)

     #:color "yellow"
     #:damage 25
     #:fire-mode 'spread)))
 
(starwars-game #:weapon-list (list (b-droid)))

7.20 Blaster Droid Kata (difficulty = 5)

read

code
#lang battlearena-starwars
 
(define (b-droid)
   (custom-weapon

7.22 Lightsaber Armor Kata (difficulty = 4)

read
Make a game that has a fully customized Lightsaber Armor.
code
#lang battlearena-starwars
 
(define (l-armor)
   (custom-armor
    #:name "Lightsaber Armor"
    #:protects-from "Lightsaber"
    #:sprite (make-icon "LA")
    #:change-damage (subtract-by 30)
    #:rarity 'rare))
 
(starwars-game #:item-list (list (l-armor)))

7.23 Lightsaber Armor Kata (difficulty = 6)

read
Make a game that has an enemy with a lightsaber and armor to protect your rebel.
code
#lang battlearena-starwars
 
(define (l-armor)
   (custom-armor
    #:name "Lightsaber Armor"
    #:protects-from "Lightsaber"
    #:sprite (make-icon "LA")
    #:change-damage (subtract-by 30)
    #:rarity 'rare))
 
(starwars-game
  #:imperial-list (list
   (custom-imperial
    #:sprite darthvader-sprite
    #:weapon (lightsaber #:color "red")))
  #:item-list (list (l-armor)))
