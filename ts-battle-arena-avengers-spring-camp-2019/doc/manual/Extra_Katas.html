9 Extra Katas

9.1 Shield Kata (difficulty = 2)

Tip: You can also try set-shield-to.

read
Make a game that has an powerup that increases your shield by 50.
code
#lang battlearena-avengers
 
(avengers-game
  #:item-list (list
   (custom-item
    #:name "Shield Powerup"
    #:sprite (make-icon "SP" 'blue 'white)
    #:on-use (change-shield-by 50))))

9.2 Shield Kata (difficulty = 2)

Tip: You can also try change-shield-by.

read
Make a game that has an powerup that recovers your shield to 100.
code
#lang battlearena-avengers
 
(avengers-game
  #:item-list (list
   (custom-item
    #:name "Max Shield Powerup"
    #:sprite (make-icon "MSP" 'blue 'white)
    #:on-use (set-shield-to 100))))

9.3 Shield Kata (difficulty = 7)

read
Make a game that has both a Shield and Max Shield defined powerup.
code
#lang battlearena-avengers
 
(define (shield-powerup)
   (custom-item
    #:name "Shield Powerup"
    #:sprite (make-icon "SP" 'blue 'white)
    #:on-use (change-shield-by 50)
    #:rarity 'uncommon
    #:respawn? #t))
 
(define (max-shield)
   (custom-item
    #:name "Max Shield Powerup"
    #:sprite (make-icon "MSP" 'blue 'white)
    #:on-use (set-shield-to 100)
    #:rarity 'epic))
 
(avengers-game
  #:item-list (list (shield-powerup) (max-shield)))

9.4 Force Field Kata (difficulty = 2)

Tip: The basic force-field will block enemy projectiles. It will also block your own projectiles.

read
Make a game that has an item with a custom name that spawns a force field.
code
#lang battlearena-avengers
 
(avengers-game
  #:item-list (list
   (custom-item
    #:name "Force Field"
    #:on-use (spawn (force-field)))))

9.5 Force Field Kata (difficulty = 3)

Tip: The default duration is 500.

read
Make a game that has an item with a custom name and sprite, that spawns a force field with a custom duration.
code
#lang battlearena-avengers
 
(avengers-game
  #:item-list (list
   (custom-item
    #:name "Force Field"
    #:sprite (make-icon "FF")
    #:on-use (spawn (force-field #:duration 1000)))))

9.6 Force Field Kata (difficulty = 4)

Tip: Setting #:allow-friendly-dart? to #t will allow you to shoot through your force-field.

read
Make a game that has an item with a custom name and sprite, that spawns a force field with a custom duration, and that allows friendly darts to go through it.
code
#lang battlearena-avengers
 
(define (force-field-item)
   (custom-item
    #:name "Force Field"
    #:sprite (make-icon "FF")
    #:on-use (spawn
     (force-field
      #:allow-friendly-dart? #t
      #:duration 1000))))
 
(avengers-game
  #:item-list (list (force-field-item)))

9.7 Droid Kata (difficulty = 2)

read
Make a game that lets you spawn a Droid that attacks at enemies.
code
#lang battlearena-avengers
 
(avengers-game
  #:power-list (list
   (custom-power
    #:name "Droid"
    #:icon (make-icon "D" 'yellow)
    #:dart (energy-droid))))

9.8 Droid Kata (difficulty = 4)

read
Make a game that lets you spawn a Droid that attacks with yellow energy blasts.
code
#lang battlearena-avengers
 
(define (my-droid)
   (custom-power
    #:name "Droid"
    #:icon (make-icon "D" 'yellow)
    #:dart (energy-droid
     #:color 'yellow
     #:damage 25
     #:fire-rate 2
     #:fire-mode 'spread)))
 
(avengers-game #:power-list (list (my-droid)))

9.14 Lava Pit Kata (difficulty = 3)

Tip: Default #:damage is 10 and default #:size is 1.

read

code
#lang battlearena-avengers
 
(avengers-game
  #:power-list (list
   (custom-power
    #:name "Lava Pit"
    #:sprite (make-icon "LP" 'red)
    #:dart (lava-builder #:damage 25 #:size 2))))

9.15 Lava Pit Kata (difficulty = 5)

Tip: Default #:range is 5.

read
Make a game that has a custom weapon that deploys a lava pit with custom damage, size, sprite, and range.
code
#lang battlearena-avengers
 
(define (my-lava-pit)
   (custom-power
    #:name "Lava Pit"
    #:sprite (make-icon "LP" 'red 'white)
    #:dart (lava-builder
     #:damage 25
     #:size 1
     #:sprite (square 10 'solid 'black)
     #:range 10)))
 
(avengers-game #:power-list (list (my-lava-pit)))

9.18 Spike Mine Kata (difficulty = 4)

Tip: Default #:damage is 2.

read
Make a game that has a weapon that deploys a spike mine with a large range and speed, high damage, and custom sprite.
code
#lang battlearena-avengers
 
(avengers-game
  #:power-list (list
   (custom-power
    #:name "Spike Mine"
    #:sprite (make-icon "SM" 'gray 'white)
    #:dart (spike-mine-builder
     #:sprite (set-sprite-color 'red spike-mine-sprite)
     #:damage 100
     #:speed 10
     #:range 50))))
