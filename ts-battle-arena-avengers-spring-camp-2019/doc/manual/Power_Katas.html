5 Power Katas

5.1 Power Kata (difficulty = 1)

Tip: Other darts besides (energy-blast) are (star-bit), (magic-orb), and (flying-hammer).

read
Make a game with an energy blast power.
code
#lang battlearena-avengers
 
(avengers-game
  #:power-list (list (energy-blast)))

5.2 Power Kata (difficulty = 1)

read
Make a game with a custom power and customize color and the damage.
code
#lang battlearena-avengers
 
(avengers-game
  #:power-list (list
   (energy-blast #:color 'orange #:damage 20)))

5.3 Power Kata (difficulty = 3)

Tip: The rarity levels are ’common, ’uncommon, ’rare, ’epic, ’legendary.

read
Make a game with a defined and customized power.
code
#lang battlearena-avengers
 
(define (my-power)
   (star-bit
    #:color 'orange
    #:damage 10
    #:speed 15
    #:rarity 'legendary))
 
(avengers-game #:power-list (list (my-power)))

5.4 Power Kata (difficulty = 5)

read
Make a game with two custom power.
code
#lang battlearena-avengers
 
(define (my-power-1)
   (magic-orb
    #:damage 10
    #:speed 15
    #:fire-mode 'spread
    #:rapid-fire? #f))
 
(define (my-power-2)
   (hammer
    #:color 'red
    #:damage 15
    #:speed 10
    #:rarity 'rare))
 
(avengers-game
  #:power-list (list (my-power-1) (my-power-2)))
