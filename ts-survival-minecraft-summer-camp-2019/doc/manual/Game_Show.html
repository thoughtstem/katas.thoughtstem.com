7 Game Show

Use these questions during Game Show time. You can ask any/all the questions in each day’s section.

7.1 Monday

  • What keys make your character move?

    • Arrow keys.

  • What is the keyword to add a character to your game?

    • #:skin

  • What is the keyword to change what your character looks like?

    • #:sprite

  • Name 3 available character sprites.

    • Anything from the Assets Library.

  • What is the default speed value?

    • 10.

7.2 Tuesday

  • How many food items will show up in the game when you use (custom-food)?

    • 1.

  • By default the food item will heal you by:

    • 10.

  • What line of code can you add to your food so that they don’t reappear in the game once you eat them?

    • #:respawn? #f.

  • Name 3 keywords available to customize the food.

    • #:sprite, #:name, #:amount-in-world, #:heals-by, #:respawn?.

  • Name 3 available food sprites.

    • Anything from the Assets Library.

7.3 Wednesday

  • What is the default value of the ore?

    • 10.

  • How many ores will spawn in the game by default?

    • 10.

  • What key do you use to collect the ores?

    • Press the spacebar when you’re close to the ore.

  • Name 5 ore types from Minecraft.

    • Coal, Iron, Gold, Redstone, Lapis Lazuli, Diamond, Emerald, Nether Quartz.

  • Tell me a keyboard shortcut; a combination of keys.

    • CTRL + C, CTRL + V, CTRL + X, CTRL + A, CTRL + Z, CTRL + Y.

7.4 Thursday

  • What is the keyword to add enemies?

    • #:mob-list

  • What does AI stand for?

    • Artificial Intelligence.

  • What is the keyword to change the number of enemies in the game?

    • #:amount-in-world

  • What line of code can you add to your mob to make them only apper in night-time?

    • #:night-only? #t.

  • Name the 3 available sprites for mobs in the game.

    • creeper-sprite, skeleton-sprite, ghast-sprite.
