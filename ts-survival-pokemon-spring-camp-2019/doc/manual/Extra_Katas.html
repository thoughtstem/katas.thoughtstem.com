9.1 Stone Kata (difficulty = 2)

read
Make a game with a Pokemon and a basic collectible stone.
code
#lang survival-pokemon
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:stone-list (list (custom-stone)))

9.2 Stone Kata (difficulty = 3)

Tip: The default value of custom-coin is 10.

read
Make a game with a Pokemon and a collectible stone worth 500.
code
#lang survival-pokemon
 
(define (my-stone) (custom-stone #:value 500))
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:stone-list (list (my-stone)))

9.3 Stone Kata (difficulty = 4)

read
Make a game with a Pokemon and 20 moon stones worth 500.
code
#lang survival-pokemon
 
(define (my-stone)
   (custom-stone
    #:sprite moonstone-sprite
    #:name "Moon Stone"
    #:value 500
    #:amount-in-world 20))
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:stone-list (list (my-stone)))

9.4 Stone Kata (difficulty = 6)

read
Make a game with a Pokemon, a moon stone, and a single shiny stone that never respawns and is worth 1000.
code
#lang survival-pokemon
 
(define (my-stone)
   (custom-stone
    #:sprite moonstone-sprite
    #:name "Moon Stone"))
 
(define (my-special-stone)
   (custom-stone
    #:sprite shinystone-sprite
    #:name "Shiny Stone"
    #:value 1000
    #:amount-in-world 1
    #:respawn? #f))
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:stone-list (list (my-stone) (my-special-stone)))

9.5 Sky Kata (difficulty = 1)

Tip: By default, the length-of-day will be 2400.

read
Make a game with a Pokemon and a slow changing sky.
code
#lang survival-pokemon
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:sky (custom-sky #:length-of-day 5000))

9.6 Sky Kata (difficulty = 2)

Tip: Max darkness takes a value between 0 (no darkness) and 255 (total blackout).

read
Make a game with a Pokemon and a fast changing sky with max-darkness.
code
#lang survival-pokemon
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:sky (custom-sky
   #:length-of-day 500
   #:max-darkness 255))

9.7 Sky Kata (difficulty = 2)

read
Make a game with a Pokemon and a darkmagenta night-sky with 150 darkness.
code
#lang survival-pokemon
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:sky (custom-sky
   #:night-sky-color 'darkmagenta
   #:max-darkness 150))

9.8 Sky Kata (difficulty = 3)

read
Make a game with a Pokemon, a very short night time, and trainers that only come out at night.
code
#lang survival-pokemon
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:trainer-list (list
   (custom-trainer
    #:amount-in-world 20
    #:night-only? #t))
  #:sky (custom-sky
   #:length-of-day 2400
   #:start-of-daytime 200
   #:end-of-daytime 2200))

9.9 Level Design Kata (difficulty = 1)

Tip: This will fill the world with two types of trees.

read
Make a survival game with a forest background filled with trees.
code
#lang survival-pokemon
 
(pokemon-game
  #:town-bg (custom-town #:image FOREST-BG)
  #:enable-world-objects? #t)

9.10 Level Design Kata (difficulty = 2)

Tip: Setting hd? #t will use more detailed assets but it will take longer to load.

read
Make a survival game with a forest background filled with high definition trees.
code
#lang survival-pokemon
 
(pokemon-game
  #:town-bg (custom-town #:image FOREST-BG)
  #:other-entities (make-world-objects
   round-tree
   pine-tree
   #:hd? #t))

9.11 Level Design Kata (difficulty = 2)

Tip: make-world-objects works best with houses or trees and must only take two types. See the Assets Library for more options.

read
Make a survival game with a pink background filled with random color candy-cane-trees and snow-pine-trees.
code
#lang survival-pokemon
 
(pokemon-game
  #:town-bg (custom-town #:image PINK-BG)
  #:other-entities (make-world-objects
   candy-cane-tree
   snow-pine-tree
   #:hd? #t
   #:random-color? #t))

9.12 Level Design Kata (difficulty = 4)

Tip: With a 3x3 bg, the posn x value can be between 0 and 480 and the y between 0 and 360. Hue is any value between 0 and 360.

read
Make a survival game with 3 world objects with customized position, tile, size, and/or hue.
code
#lang survival-pokemon
 
(pokemon-game
  #:other-entities (pine-tree (posn 100 200) #:tile 0)
  (wood-house (posn 100 200) #:tile 1 #:size 0.5)
  (brick-house
   (posn 100 200)
   #:tile 2
   #:hue (random 360)))

9.15 Town Kata (difficulty = 3)

Tip: start-tile can take a number between 0 (the default) and 9.

read
Make a game with a Pokemon and a lava custom town where the Pokemon starts on tile 4.
code
#lang survival-pokemon
 
(define (my-bg)
   (custom-town
    #:image LAVA-BG
    #:rows 2
    #:columns 2))
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:town-bg (my-bg))

9.16 Town Kata (difficulty = 3)

Tip: Start-tile can be between 0 (the default) and total-tiles minus 1. For example, a 2x2 grid has 4 tiles, so the last tile is tile 3.

read
Make a survival game with an avatar and a lava background with a 2 by 2 grid that starts on tile 3.
code
#lang survival-pokemon
 
(define (my-bg)
   (custom-town
    #:image LAVA-BG
    #:rows 2
    #:columns 2
    #:start-tile 3
    #:hd? #t))
 
(pokemon-game #:town-bg (my-bg))
