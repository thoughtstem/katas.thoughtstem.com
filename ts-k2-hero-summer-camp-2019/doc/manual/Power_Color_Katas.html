Code a game with Gamora using a GREEN Magic Orb battling Nebula using a BLUE Energy Blast.
code

(start
  (gamora magic-orb green)
  ((nebula energy-blast blue)))

4.5 Hero Kata

read
Code a game with Gamora using a RED random power battling Nebula using a YELLOW power, and Mandarin using a YELLOW power.
code
#lang k2/lang/hero/powers
 
(start
  (gamora randp red)
  ((nebula randp orange) (mandarin randp yellow)))

4.6 Hero Kata

read

(start
  (starlord randp randc)
  ((rocketracoon randp randc)
   (drax randp randc)
