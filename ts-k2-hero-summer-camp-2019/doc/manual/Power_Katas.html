3.1 Hero Kata

Tip: To use a power in the game left click. It will use towards your cursor.
read
Code a game with Thor using a Hammer battling Loki.
code
#lang k2/lang/hero/powers
 
(start (thor hammer) (loki))

3.3 Hero Kata

Tip: Any hero or villian can have any power.
read
Code a game with Gamora using a Magic Orb battling Nebula using an Energy Blast.
code
#lang k2/lang/hero/powers
 
(start
  (gamora magic-orb)
  ((nebula energy-blast)))

3.6 Hero Kata

Tip: Instead of choosing a power, you can use the Question Mark to use a random power.
read
Code a game with Thor using a random power battling Loki, Malekith, Mandarin, and Nebula all using random powers.
code
#lang k2/lang/hero/powers
 
(start
  (thor randp)
  ((loki randp)
   (malekith randp)
   (mandarin randp)
   (nebula randp)))
