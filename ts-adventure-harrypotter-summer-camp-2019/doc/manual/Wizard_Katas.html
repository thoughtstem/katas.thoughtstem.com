2 Wizard Katas

2.1 Wizard Kata (difficulty = 1)

read
Code a game with a fast Harry Potter wizard.
code
#lang adventure-harrypotter
 
(harrypotter-game
  #:wizard (basic-wizard
   #:sprite harrypotter-sprite
   #:speed 20))

2.2 Wizard Kata (difficulty = 3)

Tip: Look for more sprites in the Assets Library.

read
Code a game with a strong, fast, Harry Potter wizard.
code
#lang adventure-harrypotter
 
(define (my-wizard)
   (basic-wizard
    #:sprite harrypotter-sprite
    #:speed 20
    #:health 200
    #:max-health 200))
 
(harrypotter-game #:wizard (my-wizard))

2.3 Wizard Kata (difficulty = 2)

Tip: The default speed is 10.

read
Code a game with a fast wizard and an intro cutscene. Customize the cutscene to have a sprite and some text.
code
#lang adventure-harrypotter
 
(harrypotter-game
  #:wizard (basic-wizard
   #:sprite harrypotter-sprite
   #:speed 20)
  #:intro-cutscene (basic-cutscene
   (page
    harrypotter-sprite
    "This is the story of"
    "Harry Potter.")))

2.4 Wizard Kata (difficulty = 2)

Tip: Each string is a line of text in a cutscene.

read
Code a game with a wizard and an intro cutscene. Customize the cutscene with 2 pages, include sprites and text.
code
#lang adventure-harrypotter
 
(harrypotter-game
  #:wizard (basic-wizard #:sprite harrypotter-sprite)
  #:intro-cutscene (basic-cutscene
   (page "A long time ago," "In a magical place.")
   (page
    harrypotter-sprite
    "There was a young wizard"
    "named Harry Potter.")))
