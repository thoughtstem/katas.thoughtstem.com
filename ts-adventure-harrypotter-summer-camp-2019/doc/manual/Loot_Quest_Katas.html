4 Loot Quest Katas

4.1 Loot Quest Kata (difficulty = 6)

Tip: Every deatheater has has a #:loot-list.

read
Code a game with an NPC who has a loot quest to retrieve their stolen item. Collect the item by defeating the deatheater who stole it.
code
#lang adventure-harrypotter
 
(define stolen-chest (basic-item))
 
(harrypotter-game
  #:deatheater-list (list
   (basic-deatheater
    #:loot-list (list stolen-chest)))
  #:npc-list (list
   (basic-npc
    #:dialog (list "Someone stole my chest!")
    #:quest-list (list (loot-quest #:item stolen-chest))))
  #:spell-list (list (spear)))

4.2 Loot Quest Kata (difficulty = 8)

Tip: An item can’t talk.

read
Code a game with an NPC who has a quest to retrieve their stolen talking cat from a deatheater. Include a high-damage spell.
code
#lang adventure-harrypotter
 
(define stolen-cat
   (make-storable
    (basic-npc
     #:sprite cat-sprite
     #:dialog (list "Help me!"))))
 
(harrypotter-game
  #:deatheater-list (list
   (basic-deatheater
    #:loot-list (list stolen-cat)))
  #:npc-list (list
   (basic-npc
    #:dialog (list "Someone took my cat!")
    #:quest-list (list (loot-quest #:item stolen-cat))))
  #:spell-list (list (fireball #:damage 50)))

4.3 Loot Quest Kata (difficulty = 7)

read
Code a game with an NPC who has a rewarded quest to retrieve their stolen food. Include a wand.
code
#lang adventure-harrypotter
 
(define stolen-food
   (basic-food #:sprite apples-sprite))
 
(harrypotter-game
  #:deatheater-list (list
   (basic-deatheater
    #:loot-list (list stolen-food)))
  #:npc-list (list
   (basic-npc
    #:dialog (list "Help! Someone stole my apples!")
    #:quest-list (list
     (loot-quest
      #:item stolen-food
      #:reward-amount 400))))
  #:spell-list (list (wand)))

4.4 Loot Quest Kata (difficulty = 7)

read
Code a game with an NPC who has a quest to retrieve their stolen pumpkin. Add quest-complete dialog and a spell.
code
#lang adventure-harrypotter
 
(define stolen-pumpkin (pumpkin))
 
(define my-loot-quest
   (loot-quest
    #:item stolen-pumpkin
    #:quest-complete-dialog (list "Thank you for finding my pumpkin!")))
 
(harrypotter-game
  #:deatheater-list (list
   (basic-deatheater
    #:loot-list (list stolen-pumpkin)))
  #:npc-list (list
   (basic-npc
    #:dialog (list "Help! Someone stole my pumpkin!")
    #:quest-list (list my-loot-quest)))
  #:spell-list (list (ice-magic)))

4.5 Loot Quest Kata (difficulty = 7)

read
Code a game with an NPC who has a quest to retrieve their stolen book. Add a pickup cutscene to the stolen book.
code
#lang adventure-harrypotter
 
(define stolen-item
   (basic-item
    #:sprite flyingbook-sprite
    #:on-store (spawn
     (page
      flyingbook-sprite
      "This must be the stolen book."))))
 
(harrypotter-game
  #:deatheater-list (list
   (basic-deatheater
    #:loot-list (list stolen-item)))
  #:npc-list (list
   (basic-npc
    #:dialog (list "That thief took my book!")
    #:quest-list (loot-quest #:item stolen-item)))
  #:spell-list (list (fire-magic)))
