3 Potion Katas

3.1 Potion Kata (difficulty = 2)

Tip: The default potion heals by 10.

read
Code a game with several strong healing potions.
code
#lang adventure-harrypotter
 
(harrypotter-game
  #:potion-list (list
   (basic-potion
    #:name "Heal Potion"
    #:amount-in-world 10
    #:heals-by 20)))

3.2 Potion Kata (difficulty = 4)

Tip: Here is a link to all available colors: colors. Mix it up a bit!

read
Code a game with 2 types of potions: a common, green, weak healing potion; and a very rare, orange, powerful healing potion.
code
#lang adventure-harrypotter
 
(define my-potion
   (basic-potion
    #:name "Heal"
    #:color 'green
    #:amount-in-world 10))
 
(define special-potion
   (basic-potion
    #:name "Ultra Heal"
    #:color 'orange
    #:heals-by 50
    #:respawn? #f))
 
(harrypotter-game
  #:potion-list (list my-potion special-potion))

3.3 Potion Kata (difficulty = 4)

read
Code a game with a wizard who starts at half health. Add a powerful yellow potion that spawns a cutscene on pickup. The cutscene includes the potion sprite and text.
code
#lang adventure-harrypotter
 
(define special-potion
   (basic-potion
    #:name "Yellow Potion"
    #:color 'yellow
    #:heals-by 100
    #:on-pickup (spawn
     (page
      (colorize-potion 'yellow)
      "This tastes bad ..."
      "Wow, I'm completely healed!"))))
 
(harrypotter-game
  #:wizard (basic-wizard #:health 100 #:max-health 200)
  #:potion-list (list special-potion))

3.4 Potion Kata (difficulty = 6)

Tip: Don’t forget the recipe!

read
Code a game with several pumpkins and a cauldron that produces a powerful pumpkin potion.
code
#lang adventure-harrypotter
 
(define pumpkin-potion
   (basic-potion
    #:name "Pumpkin Potion"
    #:color 'orange
    #:respawn? #f
    #:heals-by 50))
 
(define pumpkin-potion-recipe
   (recipe
    #:product pumpkin-potion
    #:build-time 40
    #:ingredients (list "Pumpkin")))
 
(harrypotter-game
  #:ingredient-list (list (pumpkin #:amount-in-world 10))
  #:cauldron-list (list
   (basic-cauldron
    #:recipe-list (list pumpkin-potion-recipe))))

3.5 Potion Kata (difficulty = 8)

read
Code a game with an NPC and a cauldron that produces a carrot potion. Give the NPC dialog and a quest to brew that potion.
code
#lang adventure-harrypotter
 
(define carrot-potion (basic-potion))
 
(define carrot-potion-recipe
   (recipe
    #:product carrot-potion
    #:ingredients (list "Carrot")))
 
(harrypotter-game
  #:npc-list (list
   (basic-npc
    #:dialog (list "Can you make me a potion?")
    #:quest-list (list (craft-quest #:item carrot-potion))))
  #:ingredient-list (list (carrot))
  #:cauldron-list (list
   (basic-cauldron
    #:recipe-list (list carrot-potion-recipe))))
