7 Game Show

Use these questions during Game Show time. You can ask any/all the questions in each day’s section.

7.1 Monday

  • What keys make your character move?

    • Arrow keys.

  • What is the keyword to add a character to your game?

    • #:wizard

  • What is the keyword to change what your character looks like?

    • #:sprite

  • Name 3 available character sprites.

    • Anything from the Assets Library.

  • What is the default speed value?

    • 10.

7.2 Tuesday

  • How much health does a default potion heal

    • 10.

  • What is the keyword to change how much health a potion heals?

    • #:heals-by

  • What is the keyword to change the number of potions in the game?

    • #:amount-in-world

  • What two things does the player need to make a potion?

    • Recipes and ingredients.

  • What does HD stand for when referring to images?

    • High Definition.

7.3 Wednesday

  • What is the keyword to make a quest?

    • #:quest-list

  • Describe what "loot" is.

    • Anything dropped by a deatheater.

  • Name a keyboard shortcut; a combination of keys.

    • CTRL + C, CTRL + V, CTRL + X, CTRL + A, CTRL + Z, CTRL + Y.

  • What are the keywords to trigger a cutscene?

    • spawn page

  • What does USB stand for?

    • Universal Serial Bus.

7.4 Thursday

  • How can you give an NPC random responses?

    • random-npc-response

  • Why is defining things outside of the game function considered good code?

    • It makes your code easier for someone else to understand.

  • How can you give NPCs dialog?

    • #:dialog (list ... ... ...)

  • Name two things that an NPC can do in a game.

    • Start a quest, have dialog, follow the player, be stored in the player backpack.

  • How can you change an NPC’s dialog after their quest is complete?

    • #:new-response-dialog(list ... ... ...)
