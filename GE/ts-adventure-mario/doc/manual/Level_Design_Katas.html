4.1 Level Design Kata (level-design-1)

read
Code a game with a forest level that has world objects.
code
#lang adventure-mario
 
(mario-game
  #:level (basic-level #:image FOREST-BG)
  #:enable-world-objects? #t)

4.2 Level Design Kata (level-design-2)

Tip: Check the Assets Library for trees.

read
Code a game with a candy level and 2 types of randomly-colored, high-definition trees.
code
#lang adventure-mario
 
(mario-game
  #:level (basic-level #:image PINK-BG)
  #:other-entities (make-world-objects
   candy-cane-tree
   snow-pine-tree
   #:hd? #t
   #:random-color? #t))

4.3 Level Design Kata (level-design-3)

read
Code a game with a forest level filled with bricks and fences. Make everything high-definition.
code
#lang adventure-mario
 
(mario-game
  #:level (basic-level #:image FOREST-BG #:hd? #t)
  #:other-entities (make-world-objects fence brick #:hd? #t))

4.4 Level Design Kata (level-design-4)

Tip: Big Mario is still "technically" Mario.

read
Code a game with Mario and a desert level filled with high-definition blocks and pipes. Add an intro cutscene with two pages.
code
#lang adventure-mario
 
(mario-game
  #:character (basic-character #:sprite bigmario-sprite)
  #:level (basic-level #:image DESERT-BG)
  #:intro-cutscene (basic-cutscene
   (page
    "Once upon a time"
    "there was a lone plumber.")
   (page bigmario-sprite "This is his story."))
  #:other-entities (make-world-objects block pipe #:hd? #t))

4.5 Level Design Kata (level-design-5)

read
Code a game with a big question block, a randomly-colored brick house, and a low-quality pipe. Manually set positions and tiles of all these entities.
code
#lang adventure-mario
 
(mario-game
  #:other-entities (question-block
   (posn 100 200)
   #:tile 1
   #:size 2)
  (brick-house
   (posn 200 200)
   #:tile 2
   #:hue (random 360))
  (reduce-quality-by
   2
   (pipe (posn 100 200) #:tile 0)))
