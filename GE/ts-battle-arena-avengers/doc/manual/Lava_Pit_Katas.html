14.1 Lava Pit Kata (lava-pit-1)

read
Code a game with that has a custom weapon that deploys a lava pit.
code
#lang battlearena-avengers
 
(avengers-game #:power-list (list (lava-pit)))

14.2 Lava Pit Kata (lava-pit-2)

Tip: Default #:damage is 10 and default #:size is 1.

read
Code a game with that has a custom weapon that deploys a lava pit with custom damage and size.
code
#lang battlearena-avengers
 
(avengers-game
  #:power-list (list (lava-pit #:damage 25 #:size 2)))

14.3 Lava Pit Kata (lava-pit-3)

Tip: Default #:range is 5.

read
Code a game with that has a custom weapon that deploys a lava pit with custom damage, size, sprite, and range.
code
#lang battlearena-avengers
 
(define (my-lava-pit)
   (lava-pit
    #:damage 25
    #:size 2
    #:sprite (circle 10 'solid 'black)
    #:icon (make-icon "LAVA" 'red 'white)))
 
(avengers-game #:power-list (list (my-lava-pit)))
