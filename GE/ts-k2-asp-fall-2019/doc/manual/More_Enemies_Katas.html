6.1 Animal 021 Kata

read
Code a game with 2 levels: the 1st with a cat eating 3 a GREEN apples; the 2nd level with a cat eating 5 YELLOW apples and healing 3 wolves.
code
#lang k2/lang/animal/enemies
 
(start cat ((apple green 5)))
 
(start cat ((apple yellow 5)) ((wolf 3)))

6.2 Animal 022 Kata

read
Code a game with 2 levels: the 1st with a BLUE horse eating 8 peppers and healing a rabbit; the 2nd level with an ORANGE horse eating 2 peppers, healing 3 rabbits and avoiding a dog.
code
#lang k2/lang/animal/enemies
 
(start (horse blue) ((pepper 8)) (rabbit))
 
(start
  (horse orange)
  ((pepper 2))
  ((rabbit 3))
  (dog))

6.3 Animal 023 Kata

read
Code a game with 2 levels: the 1st with a sheep eating 2 RED bananas and 3 PURPLE kiwis, healing a sheep, a BLUE wolf, and 2 RED turkeys, and avoiding an enemy; the 2nd level with an animal eating 2 different foods, healing 3 different friends, and avoiding 4 different enemies
code
#lang k2/lang/animal/enemies
 
(start
  sheep
  ((banana red 2) (kiwi purple 3))
  (sheep (wolf blue) (turkey red 2))
  (rand))
 
(start
  rand
  (rand rand)
  (rand rand rand)
  (rand rand rand rand))

6.5 Animal 025 Kata

read
Code a game with 3 levels: the 1st with a goat eating 2 YELLOW mushrooms; the 2nd level with a goat eating bananas and healing 4 GREEN wolves; and the 3rd level with a goat eating apples, healing a sheep and avoiding a 3 RED dogs.
code
#lang k2/lang/animal/enemies
 
(start goat (mushroom yellow 2))
 
(start goat (banana) (wolf green 4))
 
(start goat (apple) (sheep) (dog red 3))
