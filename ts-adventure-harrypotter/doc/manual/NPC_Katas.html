5 NPC Katas

5.1 Npc Kata (npc-1)

Tip: The NPC modes are ’still, ’follow, ’wander (the default), and ’pace.

read
Code a game a witch NPC who follows you. Give her a name and a specific tile location.
code
#lang adventure-harrypotter
 
(harrypotter-game
  #:npc-list (list
   (basic-npc
    #:sprite witch-sprite
    #:name "Witch"
    #:mode 'follow
    #:position (posn 200 200)
    #:tile 3)))

5.2 Npc Kata (npc-2)

read
Code a game 2 NPCs. Give both NPCs names and multiple lines of dialog.
code
#lang adventure-harrypotter
 
(define (polite-npc)
   (basic-npc
    #:name "Riley"
    #:dialog (list
     "Oh hello there! I'm Riley."
     "Beautiful day today!")))
 
(define (rude-npc)
   (basic-npc
    #:name "..."
    #:dialog (list
     "Woah, who are you??"
     "Nevermind -- I'm too busy."
     "Move along, now!")))
 
(harrypotter-game
  #:npc-list (list (polite-npc) (rude-npc)))

5.3 Npc Kata (npc-3)

Tip: Try taking the NPC’s response outside of the game function!

read
Code a game with a wizard and a named NPC. Give the wizard 3 questions and the NPC 3 responses.
code
#lang adventure-harrypotter
 
(define player-dialog
   (player-dialog-with
    "Jordan"
    #:dialog-list (list
     "Hello. What's your name?"
     "Are you lost?"
     "What's the meaning of life?")))
 
(define npc-response
   (list
    (list "I'm Jordan," "but you can call me J.")
    (list "Nope. Just exploring!")
    (list "That's a strange question...")))
 
(harrypotter-game
  #:wizard (basic-wizard #:components player-dialog)
  #:npc-list (list
   (basic-npc
    #:name "Jordan"
    #:dialog npc-response)))

5.4 Npc Kata (npc-4)

read
Code a game with a wizard and 2 NPCs. Give the wizard and NPCs random dialog and responses.
code
#lang adventure-harrypotter
 
(harrypotter-game
  #:wizard (basic-wizard
   #:components (random-player-dialog-with "Dakota")
   (random-player-dialog-with "Alex"))
  #:npc-list (list
   (basic-npc
    #:name "Dakota"
    #:dialog (random-npc-response))
   (basic-npc
    #:name "Alex"
    #:dialog (random-npc-response))))

5.5 Npc Kata (npc-5)

Tip: Every quest has a #:new-response-dialog variable.

read
Code a game an NPC who has a quest to fetch their lost wand. Give the NPC new dialog after the quest is complete.
code
#lang adventure-harrypotter
 
(define wand-quest
   (fetch-quest
    #:item (wand)
    #:quest-complete-dialog (list "YAY! MY WAND!")
    #:new-response-dialog (list "Thanks again!")))
 
(harrypotter-game
  #:npc-list (list
   (basic-npc
    #:name "Charlie"
    #:dialog (list "Can you find my wand?")
    #:quest-list (list wand-quest))))
