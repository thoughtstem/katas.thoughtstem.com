8 Game Show

Use these questions during Game Show time. You can ask any/all the questions in each day’s section.

8.1 Monday

  • What keys make your character move?

    • For Farm, Sea, and Zoo: Arrow keys. For Superheroes: WASD keys.

  • What is the objective of the game?

    • For Farm, Sea, and Zoo: To survive by eating food. For Superheroes: To get rid of the enemies.

  • ONLY for Farm, Sea, and Zoo: What key do you use to eat the food?

    • Press the spacebar when you’re near a food item.

  • ONLY for Superheroes: How do you use your power?

    • Left-click.

  • Name 2 character sprites.

    • Reference the Ratchet GUI.

  • Name 3 food sprites.

    • Reference the Ratchet GUI.

8.2 Tuesday

  • ONLY for Farm and Zoo: What are the values of each coin?

    • Copper is worth 1, Silver is worth 2, and Gold is worth 3.

  • ONLY for Superheroes: How do you aim your power?

    • Use your mouse pointer to set the initial direction.

  • ONLY for Farm, Sea, and Zoo: How do you spawn more than 1 food or enemy?

    • Wrap the food/enemey picture in parentheses and add a number after the picture.

  • What is the biggest key in the keyboard?

    • The spacebar.

  • What is the name of the port (hole) where you plug in a mouse?

    • USB port.

  • What does the start image look like?

    • Black square with a white arrow pointing right.

8.3 Wednesday

  • ONLY for Superheroes: How do you change the color of your power?

    • Wrap the power picture in parentheses and add a colored cirlce after the picture.

  • Name 3 sprites that you used in your games today.

    • Reference the Ratchet GUI.

  • What does the question mark image do in your game?

    • It chooses a random image and uses it in your game.

  • What is the biggest key in the keyboard?

    • The spacebar.

  • Input is a way for a human to give information to a computer, what devices do we use for this?

    • Mouse/touchpad, keyboard, microphone, etc.

8.4 Thursday

  • ONLY for Zoo: What key do you use to heal the animals?

    • The h key.

  • Name 5 sprites that you used in your games today.

    • Reference the Ratchet GUI.

  • Tell me a keyboard shortcut; a combination of keys.

    • CTRL + C, CTRL + V, CTRL + X, CTRL + A, CTRL + Z, CTRL + Y.

  • What is the visual diffence between a parentheses and a bracket?

    • A parentheses is round and the bracket has straight edges.

  • Output is a way for a computer to give a human information, what devices do we use for this?

    • Monitor/projector, speakers/headphones, printer, etc.
