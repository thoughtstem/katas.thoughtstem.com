7.30 Zoo Kata

Tip: To start one game right after another, just code it directly after the first game.
read
Code a game with 2 levels: the 1st with a zookeeper healing 4 RED, 4 BLUE, and 4 GREEN monkeys; the 2nd level with a zookeeper healing 4 ORANGE, 4 YELLOW, and 4 PURPLE giraffes.
code
#lang k2/lang/zoo/friends
 
(start
  zookeeper
  ((monkey red 4)
   (monkey blue 4)
   (monkey green 4)))
 
(start
  zookeeper
  ((giraffe orange 4)
   (giraffe yellow 4)
   (giraffe purple 4)))

code
#lang k2/lang/zoo/friends
 
(start zookeeper ((monkey 2)) ((banana 2)))
 
(start zookeeper ((penguin 4)) ((fish 4)))
 
(start zookeeper ((hippo 6)) ((apple 6)))
