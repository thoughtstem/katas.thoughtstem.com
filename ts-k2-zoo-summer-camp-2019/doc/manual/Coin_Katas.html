read

4.6 Zoo Kata

read
Write a collection of games. The 1st game has an avatar eating 2 foods and collecting 5 copper coins. The 2nd game has an avatar eating 2 foods and collecting 10 gold coins. Close one game to continue with the next.
code
#lang k2/lang/zoo/coins
 
(start rand (rand rand) ((copper 5)))
 
(start rand (rand rand) ((gold 10)))
