8 Game Show

Use these questions during Game Show time. You can ask any/all the questions in each day’s section.

8.1 Monday

  • What keys make your character move?

    • Arrow keys.

  • What is the keyword to add a character to your game?

    • #:character

  • What is the keyword to change what your character looks like?

    • #:sprite

  • Name 3 available character sprites.

    • Anything from the Assets Library.

  • What is the default speed value?

    • 10.

8.2 Tuesday

  • How do you stop something from respawning?

    • #:respawn? #f

  • Name 3 properties of a coin in the game.

    • name, value, sprite, amount-in-world, respawn?, on-pickup, tile, cutscene.

  • What is the keyword to change how much a coin is worth?

    • #:value

  • Name a coin sprite.

    • gold-coin-sprite, silver-coin-sprite, copper-coin-sprite

  • What does HD stand for when referring to images?

    • High Definition.

8.3 Wednesday

  • How do you make something in the level High Definition?

    • #:hd? #t

  • What are the keywords to make a cutscene?

    • spawn page

  • How can you set the tile of entities in a level?

    • #:tile <tile-number>

  • Name a background that you’ve used.

    • FOREST-BG, PINK-BG, DESERT-BG, SNOW-BG, LAVA-BG.

  • What does USB stand for?

    • Universal Serial Bus.

8.4 Thursday

  • What is a fetch quest?

    • A quest where you have to find something and return it.

  • How can you give an NPC multiple quests?

    • #:quest-list (list <quest1> <quest2> ... <questn>)

  • Why is defining things outside of the game function considered good code?

    • It makes your code easier to read and understand.

  • How can you stop a cutscene from happening?

    • With the #:rule keyword

  • What keyword does every fetch quest need?

    • #:item
