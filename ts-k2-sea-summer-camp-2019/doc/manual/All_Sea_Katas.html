7.19 Sea Kata

Tip: To start one game right after another, just code it directly after the first game.
read
Write a collection of games. The 1st game has an avatar eating strawberries and YELLOW potatoes, avoiding an orange fish. The 2nd game is the same, except with 5 orange fish. Close one game to continue with the next.
code
#lang k2/lang/sea/enemies
 
(start
  rand
  (strawberry (potato yellow))
  (orange-fish))
 
(start
  rand
  (strawberry (potato yellow))
  ((orange-fish 5)))

7.31 Sea Kata

read
Write a collection of games. The 1st game has an avatar healing an ORANGE starfish and ORANGE crab, eating ORANGE broccoli and ORANGE kiwi, and avoiding an ORANGE shark and ORANGE jellyfish. The 2nd game has an avatar healing 3 orange fish and 3 green fish, eating 3 pineapples and 3 mushrooms, and avoiding 3 red fish and 3 yellow fish. Close one game to continue with the next.
code
#lang k2/lang/sea/friends
 
(start
  rand
  ((starfish orange) (crab orange))
  ((broccoli orange) (kiwi orange))
  ((shark orange) (jellyfish orange)))
 
(start
  rand
  ((orange-fish 3) (green-fish 3))
  ((pineapple 3) (mushroom 3))
  ((red-fish 3) (yellow-fish 3)))
