6.1 Sea Kata

read
Code a game with an animal healing a yellow fish, a BLUE shark and two other friends.
code
#lang k2/lang/sea/friends
 
(start rand (yellow-fish (shark blue) rand rand))

6.2 Sea Kata

read
Code a game with a ghost fish healing 5 PURPLE friends, eating 5 ORANGE mushrooms, and avoiding a shark.
code
#lang k2/lang/sea/friends
 
(start
  ghost-fish
  ((rand purple 5))
  ((mushroom orange 5))
  (shark))

6.3 Sea Kata

read
Code a game with 2 levels: the 1st with an animal healing a starfish and an orange fish; the 2nd level with an animal healing a crab, a green fish, a red fish, and a jellyfish.
code
#lang k2/lang/sea/friends
 
(start rand (starfish orange-fish))

6.4 Sea Kata

Tip: It is always good to write readable code.
read
Code a game with 2 levels: the 1st with an animal healing 4 GREEN octopi, eating 4 kiwi, and avoiding 4 crabs; the 2nd level with an animal healing 4 GREEN friends, eating 4 foods, and avoiding 4 wild animals.
code
#lang k2/lang/sea/friends
 
(start
  rand
  ((octopus green 4))
  ((kiwi 4))
  ((crab 4)))
 
(start
  rand
  ((rand green 4))
  ((rand 4))
  ((rand 4)))

6.5 Sea Kata

read
Code a game with an animal healing RED, ORANGE, and YELLOW friends; eating 3 foods; and avoiding GREEN, BLUE, and PURPLE wild animals.
code
#lang k2/lang/sea/friends
 
(start
  rand
  ((rand red) (rand orange) (rand yellow))
  ((rand rand rand))
  ((rand green) (rand blue) (rand purple)))

6.6 Sea Kata

read
Write a collection of games. The 1st game has an avatar healing an ORANGE starfish and ORANGE crab, eating ORANGE broccoli and ORANGE kiwi, and avoiding an ORANGE shark and ORANGE jellyfish. The 2nd game has an avatar healing 3 orange fish and 3 green fish, eating 3 pineapples and 3 mushrooms, and avoiding 3 red fish and 3 yellow fish. Close one game to continue with the next.
code
#lang k2/lang/sea/friends
 
(start
  rand
  ((starfish orange) (crab orange))
  ((broccoli orange) (kiwi orange))
  ((shark orange) (jellyfish orange)))
 
(start
  rand
  ((orange-fish 3) (green-fish 3))
  ((pineapple 3) (mushroom 3))
  ((red-fish 3) (yellow-fish 3)))
