5 Power-Up Katas

5.1 Health Kata (difficulty = 2)

Tip: You can also try set-health-to.

read
Make a game that has an powerup that increases your health by 50.
code
#lang battlearena-starwars
 
(starwars-game
  #:item-list (list
   (custom-item
    #:name "Health Powerup"
    #:sprite (make-icon "HP" 'green 'white)
    #:on-use (change-health-by 50))))

5.2 Force Field Kata (difficulty = 3)

Tip: The default duration is 500.

read
Make a game that has an item with a custom name and sprite, that spawns a force field with a custom duration.
code
#lang battlearena-starwars
 
(starwars-game
  #:item-list (list
   (custom-item
    #:name "Force Field"
    #:sprite (make-icon "FF")
    #:on-use (spawn (force-field #:duration 1000)))))

5.3 Force Field Kata (difficulty = 4)

Tip: Setting #:allow-friendly-dart? to #t will allow you to shoot through your force-field.

read
Make a game that has an item with a custom name and sprite, that spawns a force field with a custom duration, and that allows friendly darts to go through it.
code
#lang battlearena-starwars
 
(define (force-field-item)
   (custom-item
    #:name "Force Field"
    #:sprite (make-icon "FF")
    #:on-use (spawn
     (force-field
      #:allow-friendly-dart? #t
      #:duration 1000))))
 
(starwars-game
  #:item-list (list (force-field-item)))
