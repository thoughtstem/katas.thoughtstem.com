4 Weapon Katas

4.1 Lightsaber Kata (difficulty = 1)

read
Make a game that has a lightsaber.
code
#lang battlearena-starwars
 
(starwars-game #:weapon-list (list (lightsaber)))

4.2 Lightsaber Kata (difficulty = 4)

read
Make a game with your own defined fully customized lightsaber.
code
#lang battlearena-starwars
 
(define (my-lightsaber)
   (lightsaber
    #:name "Flashy"
    #:icon (make-icon "F" "blue")
    #:rarity 'rare
    #:color "blue"
    #:damage 50))
 
(starwars-game
  #:weapon-list (list (my-lightsaber)))
