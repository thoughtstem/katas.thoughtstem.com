6 Imperial Weapons Katas

6.1 Imperial Weapon Kata (difficulty = 2)

read
Make a game that has a custom imperial and give it a blaster that shoots yellow lasers.
code
#lang battlearena-starwars
 
(starwars-game
  #:imperial-list (list
   (custom-imperial
    #:weapon (blaster #:color "yellow"))))

6.2 Imperial Weapon Kata (difficulty = 2)

read
Make a game that has a custom imperial with a custom sprite and a red lightsaber.
code
#lang battlearena-starwars
 
(starwars-game
  #:imperial-list (list
   (custom-imperial
    #:sprite darthvader-sprite
    #:weapon (lightsaber #:color "red"))))

(starwars-game
