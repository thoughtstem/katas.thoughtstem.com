7.4 Environment Kata (environment-4)

read
Code a scene with a tron look where it is hard to see and has many, small hexagons with a high size and rotation variance, and a canyon like ground with a square texture.
code
#lang 3d-exploration
 
(exploration-scene
  #:speed 100
  #:environment (basic-environment
   #:preset 'tron
   #:dressing 'hexagons
   #:dressing-amount 40
   #:dressing-scale 0.5
   #:dressing-variance (variance 2 2 2)
   #:ground 'canyon
   #:ground-texture 'squares
   #:fog 0.6))

7.7 Ground Objects Kata (ground-objects-3)

read
Code a scene with a large, textured cylinder that is rotated; and a custom colored dodecahedron scaled on one axis.
code
#lang 3d-exploration
 
(define (object-1)
   (basic-cylinder
    #:radius 3
    #:height 5
    #:texture forest_bg
    #:rotation (rotation 0 1 0)))
 
(define (object-2)
   (basic-dodecahedron
    #:scale (scale 1 5 1)
    #:color (color 0 255 0)))
 
(exploration-scene
  #:ground-objects (list (object-1) (object-2)))
