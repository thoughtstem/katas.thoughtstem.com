read
Code a game with a pixelated hero.
code
#lang survival-minecraft
 
(define (my-hero)
   (basic-skin #:sprite monk-sprite))
 
(minecraft-game
  #:skin (reduce-quality-by 3 (my-hero)))

10.9 Biome Kata

Tip: start-tile can take a number between 0 (the default) and 9.

read
Code a game with a lava biome with fewer tiles.
code
#lang survival-minecraft
 
(define (my-bg)
   (basic-biome
    #:image LAVA-BG
    #:rows 2
    #:columns 2))
 
(minecraft-game #:biome (my-bg))

10.10 Biome Kata

Tip: Start-tile can be between 0 (the default) and total-tiles minus 1. For example, a 2x2 grid has 4 tiles, so the last tile is tile 3.

read
Code a game with a high-definition lava biome with fewer tiles, where the player starts on the bottom right tile.
code
#lang survival-minecraft
 
(define (my-biome)
   (basic-biome
    #:image LAVA-BG
    #:rows 2
    #:columns 2
    #:start-tile 3
    #:hd? #t))
 
(minecraft-game #:biome (my-biome))

10.17 Crafter Kata

Tip: By adding a recipe, the crafter now requires an item in the avatar’s backpack to build something.

read
Code a game with carrots, and a crafter that makes carrot stew from carrots.
code
#lang survival-minecraft
 
(minecraft-game
  #:food-list (list
   (carrot #:name "Carrot" #:amount-in-world 10))
  #:crafter-list (list
   (basic-crafter
    #:recipe-list (list carrot-stew-recipe))))

10.18 Crafter Kata

Tip: Don’t forget the recipe.

read
Code a game with fish, and a crafter that makes fish stew with high-healing power.
code
#lang survival-minecraft
 
(define (fish-stew)
   (basic-product
    #:name "Fish Stew"
    #:sprite fishstew-sprite
    #:heals-by 50))
 
(define fish-stew-recipe
   (recipe
    #:product (fish-stew)
    #:ingredients (list "Fish")))
 
(minecraft-game
  #:food-list (list (fish))
  #:crafter-list (list
   (basic-crafter
    #:recipe-list (list fish-stew-recipe))))

10.19 Crafter Kata

read
Code a game with carrots, fish, and a crafter on tile 2 (posn 200 200) that makes both carrot-stew and fish-stew.
code
#lang survival-minecraft
 
(define (fish-stew)
   (basic-product
    #:name "Fish Stew"
    #:sprite fishstew-sprite
    #:heals-by 50))
 
(define fish-stew-recipe
   (recipe
    #:product (fish-stew)
    #:ingredients (list "Fish")))
 
(define (my-cauldron)
   (basic-crafter
    #:recipe-list (list carrot-stew-recipe fish-stew-recipe)))
 
(minecraft-game
  #:food-list (list
   (carrot #:amount-in-world 10)
   (fish #:amount-in-world 10))
  #:crafter-list (list (my-cauldron)))

Tip: Setting hd? #t will use more detailed assets but it will take longer to load.

read

  #:biome (basic-biome #:image FOREST-BG)
  #:other-entities (make-world-objects
   round-tree
   pine-tree
   #:hd? #t))

Tip: make-world-objects works best with houses or trees and must only take two types. See the Assets Library for more options.

read

code
#lang survival-minecraft
 
(minecraft-game
  #:biome (basic-biome #:image PINK-BG)
  #:other-entities (make-world-objects
   candy-cane-tree
   snow-pine-tree
   #:hd? #t
   #:random-color? #t))

   (posn 100 200)

10.43 Entity Kata

Tip: The customization options for npcs are: #:sprite, #:position, #:name, #:tile, #:mode, #:game-width, #:speed, #:target, #:sound, #:scale, and #:components.

read
Code a game with 2 NPCs with custom dialog and two additional customizations each.
code
#lang survival-minecraft
 
(define (my-entity)
   (basic-entity
    #:name "Francis"
    #:tile 4
    #:dialog (list "Greetings!" "Gee, you look hungry.")))
 
(define (another-entity)
