6.1 Clicker Special 001 Kata

read
Code a forest game where the cursor is a charizard collecting firestones, and is avoiding james, and can get special ice-powers.
code
#lang clicker-pokemon-special
 
(start-forest
  charizard
  (firestone)
  (james)
  (ice-power))
